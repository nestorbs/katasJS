Kata propuesta en www.solveet.com
El sistema genera un número al azar entre 1 y 100 y el usuario deberá acertarlo. En cada intento, el sistema deberá informar al usuario si se ha pasado o si se ha quedado corto con respecto del número que debe acertar. Cuando el usuario acierte el número, el sistema deberá indicarle el número de intentos que ha necesitado para acertar
